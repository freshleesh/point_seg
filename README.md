# point_seg

Camera-LiDAR segmentation for RoboRacer using YOLO-based 2D perception fused with 3D point clouds.

## Overview

`point_seg` performs real-time sensor fusion to paint LiDAR points with semantic classes inferred from camera detections/segmentation, enabling robust environment understanding for autonomous racing scenarios.

## Highlights

- YOLO-based camera segmentation pipeline
- LiDAR point projection and semantic painting
- ROS 2 integration for real-time RoboRacer workflows
